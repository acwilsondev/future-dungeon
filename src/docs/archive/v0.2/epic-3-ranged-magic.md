# Epic 3: Combat 2.0 - Ranged & Magic

This epic moves beyond "bump combat" to include distance-based interactions.

## User Stories

### Ranged Weapons
- **As a player,** I want to use bows or throwing daggers to attack monsters from a distance, allowing for safer tactical play.

### Magic & Spells
- **As a player,** I want to find scrolls or learn spells that have various effects (e.g., Fireball, Teleport, Magic Mapping).

### Targeting System
- **As a player,** I want a dedicated "Targeting" mode where I can select a specific tile or monster for my ranged attack or spell.
- **As a developer,** I want to implement a line-drawing algorithm (like Bresenham's) to check for clear paths for projectiles.
