# Epic 8: Visual Polish & Feedback

This epic focuses on making the game feel more "juicy" and responsive.

## User Stories

### Animations
- **As a player,** I want to see a brief visual indicator (like a red flash or a '/' symbol) when a monster is hit.
- **As a player,** I want to see projectiles move across the screen when I fire an arrow or cast a spell.

### UI Transitions
- **As a player,** I want menus to fade in or pop up smoothly, and for the log to scroll clearly when many events happen at once.

### Screen Effects
- **As a player,** I want the screen to shake slightly when I take a large amount of damage, providing visceral feedback.
