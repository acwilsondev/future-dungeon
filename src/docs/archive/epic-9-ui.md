# Epic 9: User Interface (UI)

This epic focuses on the layout of information and feedback to the player.

## User Stories

### Information Display
- **As a player,** I want a dedicated area of the screen to show my character's status (HP, Level, Gold), so that I am always aware of my condition.
- **As a player,** I want a scrolling message log that describes recent events (e.g., "You hit the Orc for 5 damage"), so that I can follow the action.

### Map Clarity
- **As a player,** I want the map to be the central focus of the screen, with clear symbols for different terrain and entities.

### Menus
- **As a player,** I want to be able to open an inventory menu or a help screen without losing my place in the game, ensuring the experience is user-friendly.
