# Epic 5: Movement & Exploration

This epic covers how the player interacts with the world and how the world is revealed to them.

## User Stories

### Movement
- **As a player,** I want to move my character one tile at a time in four or eight directions, so that I can navigate the dungeon precisely.
- **As a player,** I want to be blocked by walls and other solid objects, so that the environment feels tangible.

### Field of View (FOV)
- **As a player,** I only want to see the parts of the dungeon that are within my character's line of sight, so that the game feels atmospheric and dangerous (Exploration and discovery).
- **As a player,** I want tiles I have previously seen to remain visible but dimmed (Fog of War), so that I can remember the layout of the level.

### Discovery
- **As a player,** I want to discover new monsters and items as I explore the darkness, providing a constant sense of mystery.
