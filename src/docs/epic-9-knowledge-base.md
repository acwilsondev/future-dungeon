# Epic 9: Knowledge Base & Extended UI

This epic helps the player track their progress and understand the game world.

## User Stories

### Message History
- **As a player,** I want to be able to open a full-screen view of the message log to read back through earlier events I might have missed.

### Bestiary
- **As a player,** I want to see a list of all monster types I've encountered, along with their stats and a brief description.

### Item Tooltips
- **As a player,** I want more detailed descriptions of items in my inventory, including exactly how much they heal or what bonuses they provide.
