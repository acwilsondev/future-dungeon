# Main Menu

This document outlines the design of the initial main menu for v0.6.

- ASCII art game title `FUTURE DUNGEON`
- New Game
- Load Game
- Exit
- Bottom Text: Created by Aaron Wilson
- Animated background (we'll do a starry parallax to start)
