# Epic 2: Advanced Map Features

This epic enhances the variety and interactivity of the dungeon environment.

## User Stories

### Thematic Variation
- **As a player,** I want to encounter different types of levels (e.g., limestone caves, marble halls, overgrown ruins), so that exploration feels varied.
- **As a developer,** I want to implement different generation algorithms (like Cellular Automata for caves) selectable by the builder.

### Vaults & Prefabs
- **As a player,** I want to find specially designed rooms with unique layouts and guaranteed loot (Vaults), adding "flavor" to the random generation.

### Interactive Terrain
- **As a player,** I want to interact with doors (opening/closing) and avoid hidden traps, adding tactical depth to movement.
