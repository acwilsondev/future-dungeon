# Epic 7: Identification & Alchemy

This epic adds mystery and resource management through unidentified items and basic crafting.

## User Stories

### Unidentified Loot
- **As a player,** I want to find potions and scrolls with names like "Milky Potion" or "Scrawled Scroll" whose effects are unknown until used or identified.
- **As a player,** I want to use "Identification Scrolls" to safely reveal the properties of items in my inventory.

### Alchemy Stations
- **As a player,** I want to find alchemy benches where I can combine two potions to create a more powerful effect or a different type of potion.

### Risk vs Reward
- **As a player,** I want to occasionally encounter "Cursed" items that have negative effects but are difficult to remove without special scrolls.

## Developer Goals
- Implement a "Knowledge" system that tracks which item types the player has identified.
- Create a simple recipe-based crafting UI for the Alchemy Station.
