# Epics Overview: RustLike v0.5 - "The Living World"

This version focuses on making the dungeon feel like a varied, rhythmic environment with distinct stages and powerful developer tools.

## 1. Dungeon Milestones (Rhythm)

Implement special floor types that appear at regular intervals to provide a predictable but exciting progression loop (Merchants, Altars, Bosses, Reset Shrines).

## 2. Biomes & Environmental Variety

Introduce themed dungeon segments (Crypts, Caves, Temples) with unique generation algorithms (Cellular Automata), monsters, and lighting styles.

## 3. Debug Console

Implement an in-game console for developers to spawn items, manipulate stats, teleport, and test game mechanics in real-time.
